# SkillShareTrinket
These are the code samples for Lowell Makes Skill Share 2026 Electronic Trinkets class. These use Circuit Python on the RP2040-Zero.
The RP2040-Zero board uses the GP-16 pin for the onboard WS2812 smart RGB LED. 

The CircuitPython interpreter expects to find your program in code.py. A separate folder is used for each program to avoid name collisions.
Enjoy!
